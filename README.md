# Comparative effectiveness of metabolic bariatric surgery versus incretin-based and medical therapies for weight loss and glycemic outcomes

This repository contains the R code and study-level data used for the systematic review and meta-analysis:

"Comparative effectiveness of metabolic bariatric surgery versus incretin-based and medical therapies for weight loss and glycemic outcomes: a systematic review and meta-analysis"

## Software

* R version 4.3.3
* meta package
* metafor package

## Contents

* bariatric_meta_analysis.R: R script containing the extracted study-level dataset and code used to generate the meta-analysis and forest plots.
